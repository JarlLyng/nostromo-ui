# Getting Started

Welcome to Nostromo UI! This guide helps you get started with our UI library.

## Prerequisites

- **Node.js**: >= 20.0.0
- **React**: >= 18.0.0
- **TypeScript**: >= 5.0.0 (recommended)
- **Tailwind CSS**: >= 3.0.0 or >= 4.0.0

## Installation

### From npm

Install Nostromo UI in your project:

```bash
# npm
npm install @jarllyng/ui-core @jarllyng/ui-marketing @jarllyng/ui-tw

# pnpm
pnpm add @jarllyng/ui-core @jarllyng/ui-marketing @jarllyng/ui-tw

# yarn
yarn add @jarllyng/ui-core @jarllyng/ui-marketing @jarllyng/ui-tw
```

### From GitHub (development/local)

For local development, you can clone the repository:

```bash
git clone https://github.com/JarlLyng/nostromo-ui.git
cd nostromo-ui
pnpm install
pnpm build
```

### Peer Dependencies

Nostromo UI requires the following peer dependencies:

```bash
npm install react react-dom
npm install -D tailwindcss
```

## Theming Setup

### 1. Import Base CSS

```ts
// In your entry file (e.g. main.tsx)
import "@jarllyng/ui-tw/base.css";
import "@jarllyng/ui-tw/themes/nostromo.css"; // choose or customize theme
```

### 2. Apply Theme

```html
<html data-theme="nostromo" data-color-scheme="light">
  <!-- Your content -->
</html>
```

### 3. Customize Brand Colors (Optional)

```css
[data-theme="mybrand"] {
  --color-brand-500: 220 100% 50%; /* Your brand blue */
  --color-brand-600: 220 100% 40%; /* Darker variant */
}
```

## Basic Usage

```jsx

export default function MyComponent() {
  return <Button>Click me</Button>;
}
```

## Tailwind CSS Configuration

Nostromo UI uses Tailwind CSS. Configure Tailwind in your project:

```js
// tailwind.config.js
const nostromoPreset = require("@jarllyng/ui-tw/tailwind.preset.js");

module.exports = {
  content: [
    "./src/**/*.{js,ts,jsx,tsx,mdx}",
    "./node_modules/@jarllyng/**/*.{js,ts,jsx,tsx}",
  ],
  presets: [nostromoPreset],
};
```

### Project Setup Examples

**Next.js (App Router):**

```tsx
// app/layout.tsx
import "@jarllyng/ui-tw/base.css";
import "@jarllyng/ui-tw/themes/nostromo.css";

export default function RootLayout({ children }) {
  return (
    <html data-theme="nostromo" data-color-scheme="light">
      <body>{children}</body>
    </html>
  );
}
```

**Vite:**

```tsx
// main.tsx
import "@jarllyng/ui-tw/base.css";
import "@jarllyng/ui-tw/themes/nostromo.css";
import React from "react";
import ReactDOM from "react-dom/client";
import App from "./App";

ReactDOM.createRoot(document.getElementById("root")!).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
);
```

**Dark Mode Toggle:**

```tsx
// Theme toggle example
function ThemeToggle() {
  const [isDark, setIsDark] = useState(false);

  useEffect(() => {
    document.documentElement.setAttribute(
      "data-color-scheme",
      isDark ? "dark" : "light",
    );
  }, [isDark]);

  return (
    <button onClick={() => setIsDark(!isDark)}>
      {isDark ? "Light" : "Dark"} Mode
    </button>
  );
}
```
