# Tooltip Component

The Tooltip component provides helpful text when users hover over an element.

## Example

```jsx
import { Tooltip, TooltipContent, TooltipProvider, TooltipTrigger } from '@nostromo/ui-core'

export default function Example() {
  return (
    <TooltipProvider>
      <Tooltip>
        <TooltipTrigger>Hover over me</TooltipTrigger>
        <TooltipContent>
          <p>This is a tooltip</p>
        </TooltipContent>
      </Tooltip>
    </TooltipProvider>
  )
}
```

## Accessibility

### Keyboard Navigation
- **Tab**: Moves focus to the trigger element
- **Enter/Space**: Opens the tooltip
- **Escape**: Closes the tooltip

### ARIA Attributes
- **role="tooltip"**: Identifies the element as a tooltip
- **aria-describedby**: Links the tooltip to its trigger

### Focus Management
- **Focus indicators**: Clear visual focus states
- **Screen reader support**: Proper announcements

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | Tooltip content |
| `className` | `string` | - | Additional CSS classes |
