  return (

## Props

| Prop | Type | Default | Beskrivelse |
|------|------|---------|-------------|
| `checked` | `boolean` | `false` | Om checkbox er markeret |
| `disabled` | `boolean` | `false` | Om checkbox er deaktiveret |
| `onCheckedChange` | `function` | - | Change handler |
