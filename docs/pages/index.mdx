# Nostromo UI

## Quick Install

```bash
# npm
npm install @nostromo/ui-core @nostromo/ui-marketing @nostromo/ui-tw

# pnpm
pnpm add @nostromo/ui-core @nostromo/ui-marketing @nostromo/ui-tw

# yarn
yarn add @nostromo/ui-core @nostromo/ui-marketing @nostromo/ui-tw
```
