# Nostromo UI

## Quick Install

```bash
# npm
npm install @jarllyng/ui-core @jarllyng/ui-marketing @jarllyng/ui-tw

# pnpm
pnpm add @jarllyng/ui-core @jarllyng/ui-marketing @jarllyng/ui-tw

# yarn
yarn add @jarllyng/ui-core @jarllyng/ui-marketing @jarllyng/ui-tw
```

For local development, clone the repository:

```bash
git clone https://github.com/JarlLyng/nostromo-ui.git
cd nostromo-ui
pnpm install
```
