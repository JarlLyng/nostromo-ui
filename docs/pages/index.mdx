# Nostromo UI

Velkommen til Nostromo UI - et omfattende UI bibliotek bygget med React, TypeScript og Tailwind CSS.

## Komponenter

Udforsk vores samling af komponenter:

- **Button** - Knapper med forskellige varianter
- **Input** - Input felter med validering
- **Dialog** - Modaler og dialogs
- **Badge** - Status badges og labels
- **Accordion** - Udvidbare sektioner
- **Alert** - Notifikationer og advarsler
- **Avatar** - Bruger avatars
- **Card** - Indholdskort
- **Checkbox** - Checkbox komponenter
- **Icon** - Ikoner og symboler
- **Label** - Form labels
- **Pagination** - Sideinddeling
- **Progress** - Fremskridtsindikatorer
- **Select** - Dropdown menuer
- **Separator** - Adskillelseslinjer
- **Skeleton** - Loading placeholders
- **Switch** - Toggle switches
- **Table** - Datatabeller
- **Tabs** - Tab navigation
- **Textarea** - Flerlinje tekstfelter
- **Toast** - Popup notifikationer
- **Tooltip** - Hjælpetekst

## Kom i Gang

Se [Getting Started](/getting-started) for at komme i gang med Nostromo UI.

## Storybook

Alle komponenter har interaktive eksempler i Storybook. Start Storybook for at se live eksempler:

```bash
npm run storybook
```

**Live Storybook:** [http://localhost:6007](http://localhost:6007) (når Storybook kører)
