# Storybook

Udforsk alle komponenter interaktivt i vores Storybook miljø.

## Start Storybook

For at starte Storybook, kør følgende kommando i projektets rod:

```bash
npm run storybook
```

Storybook vil køre på [http://localhost:6007](http://localhost:6007)

## Hvad er Storybook?

Storybook er et værktøj til at bygge UI komponenter og sider i isolation. Det hjælper med:

- **Komponent udvikling** - Byg komponenter i isolation
- **Dokumentation** - Automatisk genereret dokumentation
- **Testing** - Test komponenter i forskellige states
- **Design system** - Hold konsistent design

## Komponenter i Storybook

Alle vores komponenter har stories i Storybook:

- **Button** - Alle varianter og størrelser
- **Input** - Forskellige typer og states
- **Dialog** - Modaler og overlays
- **Card** - Layout komponenter
- **Table** - Datatabeller
- **Form** - Form komponenter
- **Navigation** - Menu og navigation
- **Feedback** - Alerts, toasts, progress

## Interaktive Eksempler

I dokumentationen kan du se live eksempler af komponenterne direkte integreret fra Storybook. Disse eksempler opdateres automatisk når du ændrer komponenterne.

## Tips

- **Kopier kode** direkte fra Storybook eksemplerne
- **Test forskellige props** i Storybook controls
- **Se alle varianter** på ét sted
- **Debug komponenter** i isolation
