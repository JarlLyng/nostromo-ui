Using Themes

To use a theme in your project, simply import the theme CSS file:

// Import the theme you want to use
import '@nostromo/ui-tw/themes/nostromo.css'
// or
import '@nostromo/ui-tw/themes/mother.css'
// or
import '@nostromo/ui-tw/themes/lv-426.css'
// or
import '@nostromo/ui-tw/themes/sulaco.css'

You can also switch themes dynamically by setting the data-theme attribute:

// Set theme on document element
document.documentElement.setAttribute('data-theme', 'nostromo')
document.documentElement.setAttribute('data-theme', 'mother')
document.documentElement.setAttribute('data-theme', 'lv-426')
document.documentElement.setAttribute('data-theme', 'sulaco')

Customizing Themes

Each theme uses CSS custom properties that you can override:

/* Override theme colors */
:root {
  --color-primary: 262 84% 52%;
  --color-secondary: 262 84% 45%;
  --color-accent: 142 76% 36%;
}

/* Or use HSL values directly */
:root {
  --color-primary: hsl(262 84% 52%);
  --color-secondary: hsl(262 84% 45%);
  --color-accent: hsl(142 76% 36%);
}

Theme Features

  • Consistent Design: All themes follow the same design system
  • Accessibility: Each theme meets WCAG 2.1 AA contrast requirements
  • Dark Mode: All themes support both light and dark modes
  • Customizable: Easy to override colors and variables
  • Performance: Optimized CSS with minimal bundle size
