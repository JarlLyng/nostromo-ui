Badge

Small status indicators and labels for various use cases. Perfect for notifications, status, and categorization.

Installation

import { Badge } from '@nostromo/ui-core'

Basic Usage

The Badge component provides a simple way to display status information and labels.

import { Badge } from '@nostromo/ui-core'

export default function StatusBadges() {
  return (
    <div className="flex gap-2">
      <Badge>New</Badge>
      <Badge variant="success">Active</Badge>
      <Badge variant="warning">Pending</Badge>
      <Badge variant="error">Error</Badge>
    </div>
  )
}
New Active Pending Error
