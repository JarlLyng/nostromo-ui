Avatar

User profile images with fallback support, different sizes, and status indicators. Perfect for user interfaces and profile displays.

Installation

import { Avatar, AvatarImage, AvatarFallback } from '@nostromo/ui-core'

Basic Usage

The Avatar component displays user profile images with fallback support.

import { Avatar, AvatarImage, AvatarFallback } from '@nostromo/ui-core'

export default function UserAvatar() {
  return (
    <Avatar>
      <AvatarImage src="/avatars/user.jpg" alt="User" />
      <AvatarFallback>JD</AvatarFallback>
    </Avatar>
  )
}

Props API

Complete reference for all Avatar component props.

Prop Type Default Description
size 'xs' | 'sm' | 'md' | 'lg' 'md' Size of the avatar
src string - Image source URL
alt string - Alt text for image
fallback string - Fallback text when image fails
className string - Additional CSS classes
