Progress

A flexible progress component with sci-fi styling and multiple variants. Perfect for loading states, system status, and Nostromo-themed interfaces.

Installation

The Progress component is part of the core UI package.

import { Progress, CircularProgress } from '@nostromo/ui-core';

Basic Usage

Use the Progress component to show completion status.

<Progress value={50} />
<Progress value={75} showLabel />
<CircularProgress value={60} showLabel />

Props API

Prop Type Default Description
value number - The current progress value
max number 100 The maximum progress value
size 'sm' | 'md' | 'lg' | 'xl' 'md' The size of the progress bar
variant 'default' | 'primary' | 'success' | 'warning' | 'error' | 'energy' | 'health' | 'alien' 'default' The visual variant of the progress bar
showLabel boolean false Whether to show the progress label
label string - Custom label for the progress bar
animated boolean false Whether to animate the progress bar
glow boolean false Whether to add a glow effect (sci-fi styling)
indeterminate boolean false Whether to show an indeterminate progress state

Accessibility

Progress components include proper accessibility features:

  • Proper ARIA attributes (role="progressbar", aria-valuenow, aria-valuemin, aria-valuemax)
  • Screen reader support with descriptive labels
  • Keyboard navigation support
  • High contrast support for all variants
  • Semantic HTML structure
<!-- Accessible progress bar -->
<Progress 
  value={75} 
  label="System Status" 
  showLabel 
  aria-label="System initialization progress" 
/>

<!-- Accessible circular progress -->
<CircularProgress 
  value={60} 
  showLabel 
  aria-label="Loading progress" 
/>
