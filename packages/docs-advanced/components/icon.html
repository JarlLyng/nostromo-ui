Icon

Comprehensive icon system with 100+ Phosphor icons. Perfect for navigation, actions, and visual communication.

Installation

import { Icon } from '@nostromo/ui-core'

Basic Usage

The Icon component provides access to 100+ Phosphor icons with consistent sizing and styling.

import { Icon } from '@nostromo/ui-core'

export default function IconExamples() {
  return (
    <div className="flex gap-4">
      <Icon name="house" />
      <Icon name="user" />
      <Icon name="gear" />
      <Icon name="magnifying-glass" />
    </div>
  )
}
