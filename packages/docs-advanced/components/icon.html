Icon

A flexible icon component using Phosphor Icons with multiple sizes, weights, and colors. Perfect for adding visual elements to your Nostromo UI components.

Installation

The Icon component is part of the core UI package and uses Phosphor Icons internally.

import { Icon } from '@nostromo/ui-core';

Basic Usage

Use the Icon component with any of the available icon names.

<Icon name="heart" />
<Icon name="star" />
<Icon name="user" />

Props API

Prop Type Default Description
name IconName - The name of the icon to display
size 'xs' | 'sm' | 'md' | 'lg' | 'xl' | '2xl' | '3xl' 'md' The size of the icon
weight 'thin' | 'light' | 'regular' | 'bold' | 'fill' | 'duotone' 'regular' The weight/style of the icon
color 'current' | 'primary' | 'secondary' | 'success' | 'warning' | 'error' | 'muted' 'current' The color of the icon
className string - Additional CSS classes

Accessibility

Icons should be used with proper accessibility considerations:

  • Use aria-label for decorative icons that convey meaning
  • Use aria-hidden="true" for purely decorative icons
  • Ensure sufficient color contrast for colored icons
  • Consider using text alongside icons for important actions
<!-- Decorative icon -->
<Icon name="heart" aria-hidden="true" />

<!-- Meaningful icon -->
<Icon name="search" aria-label="Search" />

<!-- Icon with text -->
<button>
  <Icon name="download" />
  Download
</button>
