Label

Form labels with accessibility support and proper association with form controls. Essential for accessible forms.

Installation

import { Label } from '@nostromo/ui-core'

Basic Usage

The Label component provides accessible labels for form controls.

import { Label } from '@nostromo/ui-core'

export default function FormLabel() {
  return (
    <div className="space-y-2">
      <Label htmlFor="email">Email Address</Label>
      <input id="email" type="email" className="w-full p-2 border rounded" />
    </div>
  )
}

Props API

Complete reference for all Label component props.

Prop Type Default Description
htmlFor string - ID of the associated form control
required boolean false Whether the field is required
children ReactNode - Label content
className string - Additional CSS classes
