Testimonials

Customer testimonials with ratings, avatars, and responsive grid layouts. Perfect for showcasing social proof and customer satisfaction.

Installation

import { Testimonials } from '@nostromo/ui-marketing'

Basic Usage

The Testimonials component provides a flexible way to display customer feedback and reviews.

import { Testimonials } from '@nostromo/ui-marketing'

const testimonials = [
  {
    name: "Sarah Johnson",
    title: "Product Manager",
    company: "TechCorp",
    avatar: "/avatars/sarah.jpg",
    rating: 5,
    content: "Nostromo UI has transformed our development workflow. The components are beautifully designed and incredibly easy to use."
  },
  {
    name: "Mike Chen",
    title: "Frontend Developer",
    company: "StartupXYZ",
    avatar: "/avatars/mike.jpg",
    rating: 5,
    content: "The documentation is excellent and the components are production-ready. Highly recommended!"
  }
]

export default function TestimonialsSection() {
  return (
    <Testimonials
      title="What Our Customers Say"
      subtitle="Don't just take our word for it"
      testimonials={testimonials}
    />
  )
}

Props API

Complete reference for all Testimonials component props.

Prop Type Default Description
title string - Section title
subtitle string - Section subtitle
testimonials TestimonialProps[] [] Array of testimonial objects
variant 'card' | 'quote' | 'centered' 'card' Visual style variant
size 'small' | 'medium' | 'large' 'medium' Size of the testimonials
columns number 3 Number of columns in grid
showRatings boolean true Whether to show star ratings
className string - Additional CSS classes
