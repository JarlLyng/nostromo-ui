Props API

Complete reference for all Features component props.

Prop Type Default Description
title string - Section title
subtitle string - Section subtitle
features FeatureProps[] [] Array of feature objects
variant 'grid' | 'list' | 'card' 'grid' Visual style variant
size 'small' | 'medium' | 'large' 'medium' Size of the features
columns number 3 Number of columns in grid
showIcons boolean true Whether to show feature icons
className string - Additional CSS classes
