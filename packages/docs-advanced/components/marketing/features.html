Features

Feature showcase grids with icons, hover effects, and flexible column layouts. Perfect for highlighting product capabilities and benefits.

Installation

import { Features } from '@nostromo/ui-marketing'

Basic Usage

The Features component provides a flexible way to showcase product capabilities and benefits.

import { Features } from '@nostromo/ui-marketing'

const features = [
  {
    icon: "🚀",
    title: "Lightning Fast",
    description: "Built for performance with optimized rendering and minimal bundle size."
  },
  {
    icon: "🎨",
    title: "Beautiful Design",
    description: "Carefully crafted components with attention to detail and modern aesthetics."
  },
  {
    icon: "♿",
    title: "Accessible",
    description: "WCAG 2.1 AA compliant with full keyboard navigation and screen reader support."
  },
  {
    icon: "🔧",
    title: "Customizable",
    description: "Highly flexible with extensive theming options and customization capabilities."
  }
]

export default function FeaturesSection() {
  return (
    <Features
      title="Why Choose Nostromo UI"
      subtitle="Built for modern developers"
      features={features}
    />
  )
}

Props API

Complete reference for all Features component props.

Prop Type Default Description
title string - Section title
subtitle string - Section subtitle
features FeatureProps[] [] Array of feature objects
variant 'grid' | 'list' | 'card' 'grid' Visual style variant
size 'small' | 'medium' | 'large' 'medium' Size of the features
columns number 3 Number of columns in grid
showIcons boolean true Whether to show feature icons
className string - Additional CSS classes
