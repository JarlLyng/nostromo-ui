Hero

Hero sections with customizable layouts, backgrounds, and call-to-action buttons. Perfect for landing pages and marketing sites.

Installation

import { Hero } from '@nostromo/ui-marketing'

Basic Usage

The Hero component provides a flexible foundation for creating compelling landing page headers.

import { Hero } from '@nostromo/ui-marketing'

export default function LandingPage() {
  return (
    <Hero
      title="Welcome to Nostromo UI"
      subtitle="Space-grade UI components for React"
      description="Built with modern web technologies including React, TypeScript, and Tailwind CSS."
      primaryAction={{
        label: "Get Started",
        href: "/getting-started"
      }}
      secondaryAction={{
        label: "View on GitHub",
        href: "https://github.com/jarl.l/nostromo-ui"
      }}
    />
  )
}

Props API

Complete reference for all Hero component props.

Prop Type Default Description
title string - Main heading text
subtitle string - Subtitle text
description string - Description text
variant 'default' | 'dark' | 'gradient' | 'minimal' 'default' Visual style variant
size 'small' | 'medium' | 'large' 'medium' Size of the hero section
contentAlign 'left' | 'center' | 'right' 'center' Content alignment
primaryAction ActionProps - Primary call-to-action button
secondaryAction ActionProps - Secondary call-to-action button
backgroundImage string - Background image URL
className string - Additional CSS classes
