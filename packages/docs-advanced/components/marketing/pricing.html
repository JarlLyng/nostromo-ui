Pricing

Pricing tables with yearly/monthly toggle, badges, and popular plan highlighting. Perfect for SaaS and subscription products.

Installation

import { Pricing } from '@nostromo/ui-marketing'

Basic Usage

The Pricing component provides a flexible way to display subscription plans and pricing tiers.

import { Pricing } from '@nostromo/ui-marketing'

const plans = [
  {
    name: "Starter",
    price: { monthly: 9, yearly: 90 },
    description: "Perfect for individuals and small teams",
    features: ["Up to 5 projects", "Basic support", "1GB storage"],
    cta: "Get Started"
  },
  {
    name: "Pro",
    price: { monthly: 29, yearly: 290 },
    description: "Best for growing businesses",
    features: ["Unlimited projects", "Priority support", "10GB storage"],
    cta: "Start Free Trial",
    popular: true
  }
]

export default function PricingSection() {
  return (
    <Pricing
      title="Simple, Transparent Pricing"
      subtitle="Choose the plan that's right for you"
      plans={plans}
    />
  )
}

Props API

Complete reference for all Pricing component props.

Prop Type Default Description
title string - Section title
subtitle string - Section subtitle
plans PlanProps[] [] Array of pricing plan objects
showToggle boolean true Whether to show monthly/yearly toggle
columns number 3 Number of columns in grid
className string - Additional CSS classes
