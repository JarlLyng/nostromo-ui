ErrorMessage

Error message components for form validation and user feedback. Essential for providing clear error communication to users.

Installation

import { ErrorMessage } from '@nostromo/ui-core'

Basic Usage

The ErrorMessage component displays validation errors and user feedback.

import { ErrorMessage } from '@nostromo/ui-core'

export default function FormWithError() {
  return (
    <div className="space-y-2">
      <label className="block text-sm font-medium">Email</label>
      <input type="email" className="w-full p-2 border border-red-500 rounded" />
      <ErrorMessage>Please enter a valid email address</ErrorMessage>
    </div>
  )
}

Please enter a valid email address
