Input

Flexible input components with validation and form integration. Built for accessibility and user experience.

Installation

import { Input } from '@nostromo/ui-core'

Basic Usage

The Input component provides a flexible foundation for form inputs with built-in validation support.

import { Input } from '@nostromo/ui-core'

export default function ContactForm() {
  return (
    <div className="space-y-4">
      <Input 
        placeholder="Enter your name" 
        label="Name"
      />
      <Input 
        type="email" 
        placeholder="Enter your email" 
        label="Email"
      />
    </div>
  )
}

Props API

Complete reference for all Input component props.

Prop Type Default Description
type 'text' | 'email' | 'password' | 'number' | 'tel' | 'url' | 'search' 'text' HTML input type
variant 'default' | 'filled' | 'outlined' | 'underlined' 'default' Visual style variant
size 'small' | 'medium' | 'large' 'medium' Size of the input
label string - Label text for the input
placeholder string - Placeholder text
disabled boolean false Whether the input is disabled
required boolean false Whether the input is required
error boolean false Whether the input has an error
success boolean false Whether the input is in success state
multiline boolean false Whether to render as textarea
rows number 3 Number of rows for textarea
onChange (event: ChangeEvent) => void - Change event handler
className string - Additional CSS classes
