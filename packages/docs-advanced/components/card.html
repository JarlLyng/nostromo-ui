Card

Flexible container components for content organization. Perfect for displaying information, forms, and interactive elements.

Installation

import { Card, CardContent, CardHeader, CardTitle, CardDescription } from '@nostromo/ui-core'

Basic Usage

The Card component provides a flexible container for organizing content.

import { Card, CardContent, CardHeader, CardTitle, CardDescription } from '@nostromo/ui-core'

export default function BasicCard() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Card Title</CardTitle>
        <CardDescription>Card description goes here</CardDescription>
      </CardHeader>
      <CardContent>
        <p>This is the card content area.</p>
      </CardContent>
    </Card>
  )
}
