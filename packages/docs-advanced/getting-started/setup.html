Setup

Configure your project to use Nostromo UI components with Tailwind CSS and proper theming.

Tailwind CSS Configuration

Configure Tailwind CSS to use Nostromo UI's preset and themes for optimal styling.

1. Install Tailwind CSS

# Install Tailwind CSS
pnpm add -D tailwindcss postcss autoprefixer

# Initialize Tailwind config
npx tailwindcss init -p

2. Configure tailwind.config.js

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./src/**/*.{js,ts,jsx,tsx}",
    "./app/**/*.{js,ts,jsx,tsx}",
    "./pages/**/*.{js,ts,jsx,tsx}",
    "./components/**/*.{js,ts,jsx,tsx}",
  ],
  presets: [require('@nostromo/ui-tw/tailwind.preset')],
  theme: {
    extend: {
      // Your custom theme extensions
    },
  },
  plugins: [],
}

3. Import Base Styles

// In your main CSS file (e.g., globals.css or index.css)
@import '@nostromo/ui-tw/base.css';

/* Your custom styles */
@tailwind base;
@tailwind components;
@tailwind utilities;

Theme Configuration

Set up theming for your application with light/dark mode support.

CSS Variables

Nostromo UI uses CSS variables for theming. The default theme is automatically applied:

/* Default Nostromo theme variables */
[data-theme="nostromo"] {
  --color-brand-500: 262 84% 52%;
  --color-brand-600: 262 84% 45%;
  --color-brand-700: 262 84% 38%;
  
  --color-neutral-50: 248 250 252;
  --color-neutral-100: 241 245 249;
  --color-neutral-900: 15 23 42;
  
  --radius-md: 0.5rem;
  --font-heading: "Inter", sans-serif;
}

Dark Mode Setup

Enable dark mode by adding the dark theme class to your HTML element:

// Enable dark mode
document.documentElement.setAttribute('data-color-scheme', 'dark');

// Or use class-based approach
document.documentElement.classList.add('dark');

React Application Setup

Configure your React application to use Nostromo UI components effectively.

1. Import Components

// Import individual components (recommended for tree-shaking)
import { Button } from '@nostromo/ui-core'
import { Input } from '@nostromo/ui-core'
import { Dialog } from '@nostromo/ui-core'

// Or import multiple components
import { 
  Button, 
  Input, 
  Dialog,
  Badge,
  Card 
} from '@nostromo/ui-core'

2. Basic App Structure

import React from 'react'
import { Button, Input, Card } from '@nostromo/ui-core'
import '@nostromo/ui-tw/base.css'

export default function App() {
  return (
    <div className="min-h-screen bg-gray-50">
      <Card className="p-6">
        <h1 className="text-2xl font-bold mb-4">Welcome to Nostromo UI</h1>
        <Input placeholder="Enter your name" className="mb-4" />
        <Button variant="primary">Launch Mission</Button>
      </Card>
    </div>
  )
}

Next.js Setup

Configure Next.js to work seamlessly with Nostromo UI components.

next.config.js

/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    appDir: true,
  },
  transpilePackages: ['@nostromo/ui-core', '@nostromo/ui-marketing', '@nostromo/ui-tw'],
}

module.exports = nextConfig

app/layout.tsx

import '@nostromo/ui-tw/base.css'
import './globals.css'

export default function RootLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  )
}

Vite Setup

Configure Vite for optimal development experience with Nostromo UI.

vite.config.ts

import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import path from 'path'

export default defineConfig({
  plugins: [react()],
  resolve: {
    alias: {
      '@': path.resolve(__dirname, './src'),
    },
  },
  optimizeDeps: {
    include: ['@nostromo/ui-core', '@nostromo/ui-marketing'],
  },
})
