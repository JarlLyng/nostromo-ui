Installation

Get started with Nostromo UI by installing the packages you need for your project.

Prerequisites

Before installing Nostromo UI, make sure you have the following installed:

Required

  • Node.js - Version 18.0.0 or higher
  • React - Version 18.0.0 or higher
  • TypeScript - Version 5.0.0 or higher (recommended)

Recommended

  • Tailwind CSS - For styling and theming
  • pnpm - For faster package management

Choose Your Package Manager

Nostromo UI supports all major package managers. Choose the one that fits your workflow:

pnpm (Recommended)

Fast, disk space efficient package manager

pnpm add @nostromo/ui-core

npm

Standard Node.js package manager

npm install @nostromo/ui-core

yarn

Fast, reliable, and secure dependency management

yarn add @nostromo/ui-core

Core Package

Install the core UI components package for essential building blocks:

# Install core components
pnpm add @nostromo/ui-core

# Or install specific components (tree-shaking friendly)
pnpm add @nostromo/ui-core/button
pnpm add @nostromo/ui-core/input
pnpm add @nostromo/ui-core/dialog

What's Included

  • Button, Input, Dialog, Badge, Card, Avatar
  • Tabs, Select, Label, HelperText, ErrorMessage
  • Error Boundary and Performance utilities
  • TypeScript definitions
  • Accessibility features

Marketing Package

Install the marketing components package for landing pages and marketing sites:

# Install marketing components
pnpm add @nostromo/ui-marketing

# Or install specific components
pnpm add @nostromo/ui-marketing/hero
pnpm add @nostromo/ui-marketing/testimonials
pnpm add @nostromo/ui-marketing/features
pnpm add @nostromo/ui-marketing/pricing

What's Included

  • Hero sections with customizable layouts
  • Testimonials with ratings and avatars
  • Features showcase grids
  • Pricing tables with toggles
  • Responsive design patterns

Tailwind CSS Package

Install the Tailwind CSS preset and themes for consistent styling:

# Install Tailwind preset and themes
pnpm add @nostromo/ui-tw

# Add to your tailwind.config.js
module.exports = {
  presets: [require('@nostromo/ui-tw/tailwind.preset')],
  // ... rest of your config
}

What's Included

  • Nostromo theme with CSS variables
  • Dark/light mode support
  • Custom color palette
  • Typography scales
  • Spacing and sizing utilities

Verify Installation

Test your installation by importing and using a component:

import { Button } from '@nostromo/ui-core'

export default function App() {
  return (
    <div>
      <h1>Welcome to Nostromo UI</h1>
      <Button variant="primary">Launch Mission</Button>
    </div>
  )
}

Troubleshooting

Common installation issues and their solutions:

Module not found errors

Make sure you're using the correct import path and that the package is installed in your project's node_modules.

TypeScript errors

Ensure you have TypeScript installed and that your tsconfig.json includes the node_modules/@types directory.

Styling issues

Make sure Tailwind CSS is properly configured and that you're importing the base styles from @nostromo/ui-tw.
