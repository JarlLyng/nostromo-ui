### Reddit (r/reactjs, r/webdev)

**Title:** Nostromo UI 1.0.0 - Production-ready React component library with TypeScript & Tailwind CSS

**Body:**
```
Hey r/reactjs!

I'm excited to share Nostromo UI 1.0.0 - a production-ready React component library I've been working on.

**What's included:**
- 27 core components (Button, Input, Dialog, Table, etc.)
- 6 marketing components (Hero, Features, Testimonials, Pricing, Gallery, Logo Wall)
- 4 complete themes with dark mode support
- 100% test coverage (691/691 tests)
- WCAG 2.1 AA accessibility compliance
- Full TypeScript support
- Tree-shakeable ESM + CJS builds

**Tech Stack:**
- React 18/19
- TypeScript (strict mode)
- Tailwind CSS v4
- Radix UI primitives
- Vitest for testing

**Documentation:**
- Live docs: https://jarllyng.github.io/nostromo-ui/
- Storybook: https://jarllyng.github.io/nostromo-ui/storybook-static/
- GitHub: https://github.com/JarlLyng/nostromo-ui

**Installation:**
```bash
pnpm add @nostromo/ui-core @nostromo/ui-marketing @nostromo/ui-tw
```

After extensive beta testing, we're ready for production use. Would love to hear your feedback!

What do you think? Any features you'd like to see in future versions?
```
